      Overview

      When Amazon ElastiCache returns an error response, the Ruby SDK constructs and raises an error. These errors have the following ancestors:

      • Aws::ElastiCache::Errors::ServiceError
      • Aws::Errors::ServiceError
      • RuntimeError

      You can rescue all ElastiCache errors using ServiceError:

      # rescue all errors from Amazon ElastiCache
begin
  # do stuff
rescue Aws::ElastiCache::Errors::ServiceError
  # ...
end
