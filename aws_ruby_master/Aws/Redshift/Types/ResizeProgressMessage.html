      Instance Attribute Details

      #avg_resize_rate_in_mega_bytes_per_second ⇒ Float (rw)

      The average rate of the resize operation over the last few minutes, measured in megabytes per second. After the resize operation completes, this value shows the average rate of the entire resize operation.

      #elapsed_time_in_seconds ⇒ Integer (rw)

      The amount of seconds that have elapsed since the resize operation began. After the resize operation completes, this value shows the total actual time, in seconds, for the resize operation.

      #estimated_time_to_completion_in_seconds ⇒ Integer (rw)

      The estimated time remaining, in seconds, until the resize operation is complete. This value is calculated based on the average resize rate and the estimated amount of data remaining to be processed. Once the resize operation is complete, this value will be 0.

      #import_tables_completed ⇒ Array<String> (rw)

      The names of tables that have been completely imported .

      Valid Values: List of table names.

      #import_tables_in_progress ⇒ Array<String> (rw)

      The names of tables that are being currently imported.

      Valid Values: List of table names.

      #import_tables_not_started ⇒ Array<String> (rw)

      The names of tables that have not been yet imported.

      Valid Values: List of table names

      #progress_in_mega_bytes ⇒ Integer (rw)

      While the resize operation is in progress, this value shows the current amount of data, in megabytes, that has been processed so far. When the resize operation is complete, this value shows the total amount of data, in megabytes, on the cluster, which may be more or less than TotalResizeDataInMegaBytes (the estimated total amount of data before resize).

      #status ⇒ String (rw)

      The status of the resize operation.

      Valid Values: NONE | IN_PROGRESS | FAILED | SUCCEEDED

      #target_cluster_type ⇒ String (rw)

      The cluster type after the resize operation is complete.

      Valid Values: multi-node | single-node
