      Overview

      When AWS Organizations returns an error response, the Ruby SDK constructs and raises an error. These errors have the following ancestors:

      • Aws::Organizations::Errors::ServiceError
      • Aws::Errors::ServiceError
      • RuntimeError

      You can rescue all Organizations errors using ServiceError:

      # rescue all errors from AWS Organizations
begin
  # do stuff
rescue Aws::Organizations::Errors::ServiceError
  # ...
end

      Request Context

      ServiceError objects have a #context method that returns information about the request that generated the error, including:

      • #params - The request params

      • #http_request

      • etc ...

      See ::Seahorse::Client::RequestContext for more information.
