      Overview

      This module provides a client for making API requests to Amazon Relational Database Service.

      Aws::RDS::Client

      The Client class provides one-to-one mapping for each API operation.

      rds = Aws::RDS::Client.new(region: 'us-east-1')
rds.operation_names
#=> [:add_role_to_db_cluster, :add_source_identifier_to_subscription, :add_tags_to_resource, :apply_pending_maintenance_action, ...]

      Each API operation method accepts a hash of request parameters and returns a response object.

      resp = rds.add_role_to_db_cluster(params)

      See Client for more information.

      Aws::RDS::Errors

      Errors returned from Amazon Relational Database Service are defined in the Errors module and extend RDS::Errors::ServiceError.

  # do stuff
rescue Aws::RDS::Errors::ServiceError
  # rescues all errors returned by Amazon Relational Database Service
end
