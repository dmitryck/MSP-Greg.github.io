      Overview

      Instance Attribute Details

      #certificate_arn ⇒ String (rw)

      The Amazon Resource Name (ARN) of the certificate. For more information about ARNs, see Amazon Resource Names (ARNs) and AWS Service Namespaces in the AWS General Reference.

      #created_at ⇒ Time (rw)

      The time at which the certificate was requested. This value exists only when the certificate type is AMAZON_ISSUED.

      #domain_name ⇒ String (rw)

      The fully qualified domain name for the certificate, such as www.example.com or example.com.

      #domain_validation_options ⇒ Array<Types::DomainValidation> (rw)

      Contains information about the initial validation of each domain name that occurs as a result of the RequestCertificate request. This field exists only when the certificate type is AMAZON_ISSUED.

      #failure_reason ⇒ String (rw)

      The reason the certificate request failed. This value exists only when the certificate status is FAILED. For more information, see Certificate Request Failed in the AWS Certificate Manager User Guide.

      Possible values:

      • NO_AVAILABLE_CONTACTS
      • ADDITIONAL_VERIFICATION_REQUIRED
      • DOMAIN_NOT_ALLOWED
      • INVALID_PUBLIC_DOMAIN
      • OTHER

      #imported_at ⇒ Time (rw)

      The date and time at which the certificate was imported. This value exists only when the certificate type is IMPORTED.

      #in_use_by ⇒ Array<String> (rw)

      A list of ARNs for the AWS resources that are using the certificate. A certificate can be used by multiple AWS resources.

      #issued_at ⇒ Time (rw)

      The time at which the certificate was issued. This value exists only when the certificate type is AMAZON_ISSUED.

      #issuer ⇒ String (rw)

      The name of the certificate authority that issued and signed the certificate.

      #key_algorithm ⇒ String (rw)

      The algorithm that was used to generate the key pair (the public and private key).

      Possible values:

      • RSA_2048
      • RSA_1024
      • EC_prime256v1

      #not_after ⇒ Time (rw)

      The time after which the certificate is not valid.

      #not_before ⇒ Time (rw)

      The time before which the certificate is not valid.

      #renewal_summary ⇒ Types::RenewalSummary (rw)

      Contains information about the status of ACM's managed renewal for the certificate. This field exists only when the certificate type is AMAZON_ISSUED.

      #revocation_reason ⇒ String (rw)

      The reason the certificate was revoked. This value exists only when the certificate status is REVOKED.

      Possible values:

      • UNSPECIFIED
      • KEY_COMPROMISE
      • CA_COMPROMISE
      • AFFILIATION_CHANGED
      • SUPERCEDED
      • CESSATION_OF_OPERATION
      • CERTIFICATE_HOLD
      • REMOVE_FROM_CRL
      • PRIVILEGE_WITHDRAWN
      • A_A_COMPROMISE

      #revoked_at ⇒ Time (rw)

      The time at which the certificate was revoked. This value exists only when the certificate status is REVOKED.

      #serial ⇒ String (rw)

      The serial number of the certificate.

      #signature_algorithm ⇒ String (rw)

      The algorithm that was used to sign the certificate.

      #status ⇒ String (rw)

      The status of the certificate.

      Possible values:

      • PENDING_VALIDATION
      • ISSUED
      • INACTIVE
      • EXPIRED
      • VALIDATION_TIMED_OUT
      • REVOKED
      • FAILED

      #subject ⇒ String (rw)

      The name of the entity that is associated with the public key contained in the certificate.

      #subject_alternative_names ⇒ Array<String> (rw)

      One or more domain names (subject alternative names) included in the certificate. This list contains the domain names that are bound to the public key that is contained in the certificate. The subject alternative names include the canonical domain name (CN) of the certificate and additional domain names that can be used to connect to the website.

      #type ⇒ String (rw)

      The source of the certificate. For certificates provided by ACM, this value is AMAZON_ISSUED. For certificates that you imported with ImportCertificate, this value is IMPORTED. ACM does not provide managed renewal for imported certificates. For more information about the differences between certificates that you import and those that ACM provides, see Importing Certificates in the AWS Certificate Manager User Guide.

      Possible values:

      • IMPORTED
      • AMAZON_ISSUED
