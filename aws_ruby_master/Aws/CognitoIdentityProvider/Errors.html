      Overview

      When Amazon Cognito Identity Provider returns an error response, the Ruby SDK constructs and raises an error. These errors have the following ancestors:

      • Aws::CognitoIdentityProvider::Errors::ServiceError
      • Aws::Errors::ServiceError
      • RuntimeError

      You can rescue all CognitoIdentityProvider errors using ServiceError:

      # rescue all errors from Amazon Cognito Identity Provider
begin
  # do stuff
rescue Aws::CognitoIdentityProvider::Errors::ServiceError
  # ...
end
