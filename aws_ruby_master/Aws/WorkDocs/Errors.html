      Overview

      When Amazon WorkDocs returns an error response, the Ruby SDK constructs and raises an error. These errors have the following ancestors:

      • Aws::WorkDocs::Errors::ServiceError
      • Aws::Errors::ServiceError
      • RuntimeError

      You can rescue all WorkDocs errors using ServiceError:

      # rescue all errors from Amazon WorkDocs
begin
  # do stuff
rescue Aws::WorkDocs::Errors::ServiceError
  # ...
end
