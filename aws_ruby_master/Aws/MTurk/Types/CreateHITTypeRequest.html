      Instance Attribute Details

      #assignment_duration_in_seconds ⇒ Integer (rw)

      The amount of time, in seconds, that a Worker has to complete the HIT after accepting it. If a Worker does not complete the assignment within the specified duration, the assignment is considered abandoned. If the HIT is still active (that is, its lifetime has not elapsed), the assignment becomes available for other users to find and accept.

      #auto_approval_delay_in_seconds ⇒ Integer (rw)

      The number of seconds after an assignment for the HIT has been submitted, after which the assignment is considered Approved automatically unless the Requester explicitly rejects it.

      #description ⇒ String (rw)

      A general description of the HIT. A description includes detailed information about the kind of task the HIT contains. On the Amazon Mechanical Turk web site, the HIT description appears in the expanded view of search results, and in the HIT and assignment screens. A good description gives the user enough information to evaluate the HIT before accepting it.

      #keywords ⇒ String (rw)

      One or more words or phrases that describe the HIT, separated by commas. These words are used in searches to find HITs.

      #qualification_requirements ⇒ Array<Types::QualificationRequirement> (rw)

      A condition that a Worker's Qualifications must meet before the Worker is allowed to accept and complete the HIT.

      #reward ⇒ String (rw)

      The amount of money the Requester will pay a Worker for successfully completing the HIT.

      #title ⇒ String (rw)

      The title of the HIT. A title should be short and descriptive about the kind of task the HIT contains. On the Amazon Mechanical Turk web site, the HIT title appears in search results, and everywhere the HIT is mentioned.
