      Overview

      This module provides a client for making API requests to Amazon Elastic Compute Cloud.

      Aws::EC2::Client

      The Client class provides one-to-one mapping for each API operation.

      ec2 = Aws::EC2::Client.new(region: 'us-east-1')
ec2.operation_names
#=> [:accept_reserved_instances_exchange_quote, :accept_vpc_peering_connection, :allocate_address, :allocate_hosts, ...]

      Each API operation method accepts a hash of request parameters and returns a response object.

      resp = ec2.accept_reserved_instances_exchange_quote(params)

      See Client for more information.

      Aws::EC2::Errors

      Errors returned from Amazon Elastic Compute Cloud are defined in the Errors module and extend EC2::Errors::ServiceError.

      begin
  # do stuff
rescue Aws::EC2::Errors::ServiceError
  # rescues all errors returned by Amazon Elastic Compute Cloud
end

      See Errors for more information.
