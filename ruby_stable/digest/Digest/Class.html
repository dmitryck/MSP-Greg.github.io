def self.base64digest(str, *args)
  [digest(str, *args)].pack('m0')
end

      .digest(string, *parameters) ⇒ hash_string

      Returns the hash value of a given string. This is equivalent to ::.new(*parameters).digest(string), where extra parameters, if any, are passed through to the constructor and the string is passed to #digest().

      .file(name, *args)

      Creates a digest object and reads a given file, name. Optional arguments are passed to the constructor of the digest class.

      p Digest::SHA256.file("X11R6.8.2-src.tar.bz2").hexdigest
# => "f02e3c85572dc9ad7cb77c2a638e3be24cc1b5bea9fdbb0b0299c9668475c534"
